# MultiSource-AF
MultiSource‑AF is a novel multisource adaptive fusion framework. It  integrates three key components: an adaptive upsampling  module for resolution enhancement, a cross-modal  transformer for deep HSI-SAR interaction, and a centercontext gated fusion mechanism for balancing local and 
contextual cues.
